# ESP32-S3 智能腰带项目

## 引脚分配表

| 功能 | 引脚 | 说明 |
|------|------|------|
| **超声波模块1** | | |
| TRIG1 | 1 | 超声波1触发引脚 |
| ECHO1 | 2 | 超声波1回波引脚 |
| **超声波模块2** | | |
| TRIG2 | 0 | 超声波2触发引脚 |
| ECHO2 | 38 | 超声波2回波引脚 |
| **震动马达** | | |
| M1 | 39 | 震动马达1 |
| M2 | 40 | 震动马达2 |
| M3 | 41 | 震动马达3 |
| M4 | 42 | 震动马达4 |
| M5 | 45 | 震动马达5 |
| M6 | 48 | 震动马达6 |
| **MPU6050 (I2C1)** | | |
| MPU_SDA | 47 | I2C1数据引脚 |
| MPU_SCL | 21 | I2C1时钟引脚 |
| **摄像头 (I2C0)** | | |
| CAM_SDA | 20 | I2C0数据引脚 |
| CAM_SCL | 19 | I2C0时钟引脚 |

## 注意事项

- **GPIO 26~37** 保留给 SPI Flash/PSRAM，请勿使用！
- 使用双 I2C 控制器：
  - I2C0: 用于摄像头通信
  - I2C1: 用于 MPU6050 传感器通信
